Vantagens de usar Linux

January 1, 13 min read

Antes de você se aprofundar nos motivos abaixo, gostaríamos de lembrar que esta é uma opinião dos editores baseada em nossas próprias experiências profissionais.

Se você não sabe o que é Linux, recomendamos esse curso gratuíto (sem e-mail, sem pegadinha) do Guia foca linux.

Você não precisa ser hacker, nem nada para usar linux. A maioria das distribuições Linux tem interface gráfica e é muito amigável

Ubuntu 22.10

Velocidade e Praticidade

Ao instalar o linux no seu computador, a primeira coisa que você vai notar é o sentimento de que seu computador agora é mais veloz do que era antes.

A diferença é nítida ao abrir os programas mais básicos e necessários do seu dia-a-dia como:

  • Navegador
  • Explorador de arquivos
  • Editor de texto
  • Editor de planilhas
  • Editor de código

Quantas vezes você teve que instalar um leitor de PDF? ou o famigerado Winrar para poder abrir arquivos no formato zip e que tal um anti-vírus? e então app após app o sistema ficava cada vez mais lento. Bom esqueça tudo isso, o Linux já contém a maioria dos programas necessários para o uso diário e o melhor de tudo é que a maioria deles são programas open source que são testados e analisados por milhares de usuários para garantir a excelência dos programas.

Tem mais? tem.

TODAS as distribuições Linux que já utilizamos possuí um gerenciador de pacotes que permite instalar praticamente TODOS os programas mais comuns, além disso a maioria das bibliotecas, linguagens de programação e ferramentas de desenvolvimento rodam NATIVAMENTE no Linux, até porque a maioria delas é feita no Linux 😉

Dentre os gerenciadores de pacotes os nossos preferidos são:

  • pacman
  • yay

Ambos estão disponíveis no Arch Linux, Manjaro e outros derivados do Arch.

Automação e Customização

Pessoalmente nós do Byteverso acreditamos que duas das maiores vantagens do Linux são:

  • Linguagem Bash
  • Terminal

É incrível como você pode automatizar quase qualquer coisa em uma distribuição Linux, por exemplo:

Script simples de backup de arquivos

!#/bin/bash

usuario=${USER}
pastas_para_salvar=("~/Documents", "~/Imagens", "~/Codigo")
pasta_destino="~/backup"
data=$(date +"%b_%d_%y_hora_%H_%M")

for i in "${pastas_para_salvar[@]}"; do
  sudo tar -Pczf $pasta_destino/$i-$data.tar.gz $i
  if [ $? -eq 0 ]; then
    echo "$i backup feito com sucesso!"
  else
    echo "$i backup falhou."
  fi
done

Este exemplo é realmente muito simples, com um pouco mais de conhecimento você pode fácilmente automatizar e-mails, mensagens no telegram, web-crawlers, e até mesmo…. Piadas do Chuck Norris 😂

!#/bin/bash

curl -s https://api.chucknorris.io/jokes/random | jq .value -r

Crescimento profissional

O simples ato de utilizar Linux vai te condicionar a pensar mais como um desenvolvedor e menos como um usuário, problemas acontecerão e você terá que lidar com eles. Felizmente a comunidade de utilizadores de Linux e os próprios desenvolvedores é receptiva e aberta (na maioria das vezes) e provavelmente alguém já teve o mesmo problema que você.

Você saber usar Linux é uma vantagem no mercado profissional também, pois os sistemas que você irá desenvolver possivelmente irão comunicar-se com outros sistemas que rodam aonde? Exatamente, no Linux e na AWS, então é melhor que você tenha familiaridade com o sistema não é?
